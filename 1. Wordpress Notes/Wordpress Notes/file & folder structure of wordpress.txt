your-theme-name/
├── style.css           → Required. Theme header info + main CSS.
├── index.php           → Required. Fallback template.
├── functions.php       → Theme functions, hooks, and setup.
├── screenshot.png      → Optional. Theme preview image in WP admin.
├── header.php          → Site <head> + opening <body>.
├── footer.php          → Site footer (closing tags, etc.).
├── sidebar.php         → Sidebar content (widgets, etc.).
├── page.php            → Template for static pages.
├── single.php          → Template for single posts.
├── archive.php         → Template for categories, tags, authors.
├── search.php          → Template for search results.
├── 404.php             → Template for not found pages.
├── comments.php        → Template for post comments.
├── style.css           → Main stylesheet (includes theme metadata)
└── /template-parts/    → Reusable chunks (e.g., post content layouts)
